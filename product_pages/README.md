# ClipPaste GitHub Pages

This directory contains the GitHub Pages website for ClipPaste, including all pages required for Mac App Store submission.

## Pages Included

### Required for App Store Submission

1. **[index.html](https://Phieu-Tran.github.io/ClipPaste/)** - Main landing page
   - Product overview and features
   - Screenshots (light and dark mode)
   - Keyboard shortcuts
   - Download links
   - Used as the **Marketing URL** for App Store submission

2. **[privacy.html](https://Phieu-Tran.github.io/ClipPaste/privacy.html)** - Privacy Policy
   - Comprehensive privacy policy explaining data handling
   - Required for App Store submission
   - Used as the **Privacy Policy URL**

3. **[support.html](https://Phieu-Tran.github.io/ClipPaste/support.html)** - Support Page
   - FAQ section
   - Contact information
   - Bug reporting instructions
   - System requirements
   - Used as the **Support URL** for App Store submission

4. **[terms.html](https://Phieu-Tran.github.io/ClipPaste/terms.html)** - Terms of Service
   - End User License Agreement (EULA)
   - GPL-3.0 license information
   - User responsibilities and disclaimers

## GitHub Pages Setup

To enable GitHub Pages for this site:

1. Go to **Repository Settings** → **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Select branch: **main** (or **master**)
4. Select folder: **/docs/product_pages**
5. Click **Save**

The site will be available at: `https://Phieu-Tran.github.io/ClipPaste/`

## URLs for App Store Connect

When submitting ClipPaste to the Mac App Store, use these URLs:

- **Support URL**: `https://Phieu-Tran.github.io/ClipPaste/support.html`
- **Marketing URL**: `https://Phieu-Tran.github.io/ClipPaste/`
- **Privacy Policy URL**: `https://Phieu-Tran.github.io/ClipPaste/privacy.html`

## Updating the Site

All pages use pure HTML/CSS with no build process required. Simply edit the HTML files and push to GitHub. Changes will be live within a few minutes.

## Design Features

- Responsive design for mobile and desktop
- Automatic dark mode support (follows system preferences)
- macOS-native design language
- Accessible and SEO-friendly
- No JavaScript dependencies
